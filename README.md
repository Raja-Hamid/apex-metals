# Apex Metals

Marketing site for Apex Metals — premium metal recycling for NSW. Single-page landing with doorstep-pickup positioning, services, materials list, quote form, and contact details.

## Stack

Vite + React 18 + TypeScript + Tailwind CSS v4. No backend, no router.

## Develop

```sh
npm install
npm run dev        # dev server on http://localhost:5173
npm run typecheck  # strict TS check
npm run build      # production build to dist/
npm run preview    # serve the production build
```
